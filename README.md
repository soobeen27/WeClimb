# 🧗‍♀️ WeClimb

**WeClimb**는 클라이밍 애호가들을 위한 iOS 애플리케이션입니다.  
사용자들은 클라이밍 루트를 기록하고, 다른 클라이머들과 정보를 공유할 수 있습니다.

📱 [앱스토어 링크](https://apps.apple.com/kr/app/%EC%9C%84%ED%81%B4-%ED%95%A8%EA%BB%98-%EB%A7%8C%EB%93%9C%EB%8A%94-%EB%B3%BC%EB%8D%94%EB%A7%81-%EC%9C%84%ED%82%A4/id6670149812)
## 📱 주요 기능

  다른 클라이머들과 루트 정보를 공유하고, 피드백을 주고받을 수 있습니다.
  
  <img src="https://github.com/user-attachments/assets/fb3fb13c-842f-4618-a759-d0fcec454e3f" height="500">

  다른 클라이머의 등반영상을 보고 좋아요를 하거나 댓글을 작성할 수 있습니다.

  <img src="https://github.com/user-attachments/assets/ab5368e3-bf30-40e1-b912-75d0d7c9bf86" height="500">

  암장을 검색하고 세팅을 보거나 정보를 확인할 수 있습니다.
  
  <img src="https://github.com/user-attachments/assets/60bca9b9-3873-4c90-a91e-13ff963a966b" height="500">

  내 프로필에서 내 등반기록을 확인하고, 설정에서 알림 설정, 프로필 편집, 차단목록 확인 등을 할 수 있습니다.

  <img src="https://github.com/user-attachments/assets/f1dfa59b-2cc1-4335-bc13-24b97b219828" height="500">

  암장을 선택하고 사진, 동영상들을 선택한 뒤 각 미디어에 세팅을 설정하여 게시글을 업로드 할 수 있습니다.

  <img src="https://github.com/user-attachments/assets/3f22befb-3ffb-4b5e-bb6e-bbf8da7c3729" height="500">

  메인피드 게시글에서 난이도를 클릭하거나 암장페이지에서 클릭했을때 동일한 세팅의 문제들만 볼 수 있습니다.
